# chat-nodejs-reactjs

This application needs [Node.js](https://nodejs.org/) v10+ and [yarn](https://yarnpkg.com/).

### Setup Backend
To install the dependencies:

```sh
$ cd backend
$ yarn install
```

### Run

To start the server run

```sh
$ yarn start
```

Port 3333

### Setup Frontend
To install the dependencies:

```sh
$ cd frontend
$ yarn install
```

### Run

To start the server run

```sh
$ yarn start
```

Port 3000

Documentation
http:localhost:3333/swagger
